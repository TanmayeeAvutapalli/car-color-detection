# Car Color Detection System
 Traffic detection system that identifies cars, blue cars, and people in images and videos using YOLOv8 and Gradio.

## Features
- Detects cars, blue cars,and  people
- Supports both images and videos
- Web-based GUI for easy file uploads
- Cumulative vehicle counting in videos

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python car_color_detection_gui.py
```

## Technologies
- Python
- YOLOv8
- OpenCV
- Gradio
- NumPy
